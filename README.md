# FontEndEffects
- 实现各种前端好看、酷炫、有用的一些特效。主要涉及html(5)、css(3)技术
- 基于vue-cli3构建项目整体骨架，项目核心是由各种实现的前端特效demo以及对实现各个特效涉及的核心技术进行分析的md文档构成。
- 主要涉及css(3)的新增的特效（属性）的利用。只要你会用，特效不是问题。
- 基于好看、有实际用途这两个标准
- 项目还有一个test文件用于实现特效前的练习，暂不上传github。
